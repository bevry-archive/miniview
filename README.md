## Install

### [Node](http://nodejs.org/), [Browserify](http://browserify.org/)
- Use: `require('miniview')`
- Install: `npm install --save miniview`

### [Ender](http://ender.jit.su/)
- Use: `require('miniview')`
- Install: `ender add miniview`
